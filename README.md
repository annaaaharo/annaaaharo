Sóc **Enginyera de Software Junior** a **Barcelona**, amb grau en **Enginyeria Informàtica**. Desenvolupo programari a tota la pila, des de sistemes client-servidor i programació de baix nivell fins a pipelines de machine learning i anàlisi de dades.
